About Me

I'm a systems engineer specializing in accessibility technology, with deep expertise in AI/ML, real-time transcription systems, and high-performance programming. Currently working as a Real-Time Captioning Coordinator at Rochester Institute of Technology.

Background

My work sits at the intersection of technology and human accessibility. I started in 2018 as a real-time captionist, discovering and mastering TypeWell's advanced "Extreme mode" through reverse-engineering. This technical curiosity evolved into building complete ML pipelines, Rust systems, and production applications that serve real users.

What drives me is creating technology that enables equity. Whether it's fine-tuning LLMs on academic transcripts, building sub-millisecond abbreviation engines in Rust, or deploying campus-wide equipment management systems, I focus on solutions that work in production and solve real problems.

Technical Focus

Machine Learning & NLP

Fine-tuning large language models (Llama, Qwen) with LoRA adapters, building custom NLP pipelines for transcript analysis, and developing domain-specific language models. Processed 27.4M words across 8,647 transcripts spanning 126 academic domains.

Systems Programming

Building high-performance systems in Rust with focus on memory efficiency, lock-free concurrency, and sub-millisecond latency. Expertise in arena allocators, trie data structures, and real-time processing pipelines.

Speech Recognition

Implementing and optimizing Vosk, Kaldi, and Azure STT systems for academic transcription. Building hybrid ASR pipelines that combine automatic speech recognition with intelligent text prediction and correction.

Web Applications

Full-stack development with Python (Flask, FastAPI), React, and modern web technologies. Building production systems with proper security, authentication, API integration, and real-time updates via WebSockets.
